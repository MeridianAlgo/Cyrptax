# Changelog

## v0.1 (Initial Release)
- Basic CSV normalization and tax calculation
- Support for major exchanges (Binance, Coinbase, Kraken)
- FIFO tax calculation method
- Simple CLI interface

## v0.2 (Enhanced Features)
- Added more exchanges (Gemini, KuCoin, Bitfinex, Bitstamp, etc.)
- Multiple tax methods (FIFO/LIFO/HIFO)
- Data validation and error detection
- Enhanced CLI with comprehensive options
- TurboTax compatible reporting
- PDF summary reports
- Price fetching from CoinGecko API
- Comprehensive logging system
